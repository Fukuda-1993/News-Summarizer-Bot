# AI News Summarizer Bot (AI & Programming Edition)

AI（Groq / Llama 3.3）を活用し、AI・プログラミング領域の最新技術ニュースを自動収集・要約してLINEに届ける自動化システムです。

## 概要

情報収集の効率化を目的として開発しました。Google Apps Script（GAS）をベースに、ITmediaの「AI・プログラミング」専門フィードから最新記事を抽出し、AIがビジネスマン向けに要約。リンクと共にLINE Messaging APIを通じてプッシュ通知を行います。

## 主な機能

- **技術記事の自動要約**: Groq API（Llama-3.3-70b）を使用。難解な技術記事も重要なポイント3点（各1行）で簡潔にまとめます。
- **特定ジャンル特化型**: 「AI・プログラミング」領域にターゲットを絞ることで、開発者にとってノイズの少ない高品質な情報収集を実現。
- **堅牢なパース処理**: RSSフィードのXML構造に不備（閉じタグの欠損等）がある場合でも、正規表現を用いた独自抽出ロジックにより安定して稼働します。
- **完全自動運用**: GASのトリガー機能により、毎日決まった時間に最新ニュースを自動配信します。

## 使用技術

| カテゴリ         | 技術・サービス                        |
| :--------------- | :------------------------------------ |
| **Language**     | Google Apps Script (JavaScript)       |
| **LLM / AI**     | Groq API (Llama-3.3-70b-versatile)    |
| **Notification** | LINE Messaging API                    |
| **Data Source**  | ITmedia RSS Feed (AI・プログラミング) |

## システム構成

1. **収集**: `Main.gs` が専門カテゴリのRSSをフェッチ。
2. **加工**: `XmlService` では解析不可能なイレギュラーXMLを、正規表現によるパターンマッチングで抽出しデータ損失を防止。
3. **要約**: `Groq.gs` が高速推論エンジンを用いて、技術内容を構造化要約。
4. **配信**: `line.gs` が整形されたテキスト（タイトル・要約・URL）をLINEへ送信。

## セットアップ

1. LINE DevelopersでチャネルアクセストークンとユーザーIDを取得。
2. [Groq Cloud](https://console.groq.com/)でAPIキーを取得。
3. GASの「スクリプトプロパティ」に以下のキーを設定。
   - `GROQ_API_KEY`
   - `LINE_ACCESS_TOKEN`
   - `LINE_USER_ID`

## トリガー設定

本システムは、GASのトリガー機能を利用して以下のスケジュールで自動実行されます。これにより、毎日能動的にニュースサイトを訪問する手間を省き、プッシュ型での情報収集を可能にしています。

| 実行関数 | イベントのソース | 実行タイミング              | 備考                                         |
| :------- | :--------------- | :-------------------------- | :------------------------------------------- |
| `main`   | 時間主導型       | 毎日1回（例：午前7時〜8時） | 最新ニュースの収集・要約・LINE送信を一括実行 |

## こだわりポイント（技術的工夫）

- **正規表現による堅牢なスクレイピング**:
  外部配信データのXML構文エラー（属性のクォート漏れや閉じタグ欠落など）によるシステム停止を防ぐため、標準のXMLパーサーに依存せず、正規表現を用いて確実にデータを抽出する補助関数 `extractTagContent` を自作しました。
- **セキュアな設計**:
  プロジェクト全体の設定を `Config.gs` に集約しつつ、実データはスクリプトプロパティ（環境変数）に逃がすことで、ソースコード公開時のセキュリティを確保しました。
